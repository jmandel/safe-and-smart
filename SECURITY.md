# Security policy for the spike

This repository is an architecture experiment and must not be used with real PHI.

Report suspected security defects privately to the repository owner. Include the affected commit, browser/version, deployment headers, reproduction steps, and whether any network or credential disclosure occurred.

Before a real deployment, complete the work in `docs/PRODUCTION_GAPS.md` and `docs/SECURITY_TEST_PLAN.md`, commission independent security review, and validate every supported managed-browser configuration.
